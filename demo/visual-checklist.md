# Visual Checklist

Run `npm run build`, then `npm run serve:demo` and open the demo page.

- Verify plain import gives readable defaults without extra classes or root attributes.
- Verify automatic light and dark mode switching follows system preference.
- Verify headings, paragraphs, lists, forms, tables, and media adapt across narrow and wide viewports.
- Verify the hierarchical table keeps its structure and scrolls on small screens.
- Verify the dialog opens and closes correctly.
- Verify select and optgroup remain readable in dark mode.
- Verify no UI depends on any color palette beyond foreground and background.
